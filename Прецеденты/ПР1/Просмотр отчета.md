# Просмотр отчета
**Краткое описание:** Руководитель изучает ранее сформированные отчеты.

**Основной поток:**
1. Руководитель в UI открывает раздел "Отчеты".
2. Система (через Модуль отчетности) загружает из базы данных и отображает список сохраненных отчетов (название, дата создания).
3. Руководитель выбирает отчет из списка.
4. Модуль отчетности извлекает из базы данных полные данные выбранного отчета.
5. Система отображает детализированное представление отчета (статистику, список заданий).
6. Руководитель анализирует данные отчета

**Альтернативные потоки:**
A1. **Отчет не найден:** На шаге 4 система отображает сообщение "Отчет не найден".

**Предусловия:** Руководитель авторизован, отчеты сформированы
**Постусловия:** Отчет отображен для анализа
