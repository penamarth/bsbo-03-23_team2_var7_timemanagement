# Выбор исполнителя

### Описание
Руководитель выбирает и назначает сотрудника на выполнение конкретного задания. Этот процесс включает в себя поиск подходящих кандидатов, их оценку и подтверждение выбора

### Актёры
- **Руководитель (Director)**: Выбирает задание, просматривает кандидатов, назначает исполнителя
- **Система (System)**: Формирует список кандидатов, отображает информацию, сохраняет назначение
- **Исполнитель (Executor)**: Сотрудник, назначаемый на задание

### Предусловия
- Руководитель аутентифицирован в системе
- Задание существует, создано и доступно для редактирования
- В системе зарегистрированы потенциальные исполнители

### Постусловия
- Исполнитель назначен на задание

### Основной успешный сценарий
1. Руководитель открывает раздел "Проекты"
2. Система отображает список всех проектов, доступных руководителю
3. Руководитель выбирает нужный проект
4. Система отображает список заданий внутри проекта
5. Руководитель выбирает задание, для которого требуется назначить исполнителя
6. Система отображает текущего исполнителя (если есть) и предлагает действие "Назначить исполнителя"
7. Руководитель инициирует назначение
8. Система формирует список кандидатов с краткой информацией о них, их навыках и текущих задачах
9. Руководитель выбирает исполнителя из списка
10. Система запрашивает подтверждение назначения
11. Руководитель подтверждает выбор
12. Система обновляет запись задания, устанавливает нового исполнителя
13. Система отображает руководителю обновлённую информацию о задании

### Альтернативные потоки:
- **A1. Выбран исполнитель с высокой загрузкой**
  1. Система предупреждает о высокой загрузке и запрашивает подтверждение
  2. Руководитель может подтвердить или выбрать другого исполнителя
- **B1. Отмена действия**
  На любом шаге до подтверждения руководитель нажимает «Отмена», система закрывает окно без изменений
- **C1. Ошибка сохранения / конфликт обновления**
  Если другой руководитель в это время уже назначил исполнителя, система сообщает о конфликте
  Руководитель обновляет данные и повторяет действие
